# Scrapping de versiones estenográficas de discursos presidenciales en México

Se generó el repositorio para tener las versiones estenográficas de las conferencias matutinas del presidente de México así como las de informes acerca de la epidemia de COVID-19.

ArchivosCompletos contiene únicamente dos archivos de texto, uno para las conferencias matutinas y otro para las de COVID-19 ambos tienen todas las conferencias por lo que son bastante grandes. Si se busca una fecha específica es mejor ir a las carpetas de ConferenciasMatutinas y ConferenciasCOVID.

ConferenciasMatutinas y ConferenciasCOVID contienen el texto y una imagen de nube de palabras por conferencia.
